# Just_Paint_It
## Description  
This is the improved version of [Custom_Paint_App](https://github.com/Varshithvhegde/Custom_Paint_App)  
## Features  
* Changing the stroke size  
* Changing color of the stroke  
* Saving canvas to phone storage
